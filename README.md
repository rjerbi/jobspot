# JobSpot

A simple front-end job search web application built with **HTML, CSS, and JavaScript**.  
It allows users to **sign up, log in, browse job listings, and filter jobs by keywords and location**, using **client-side logic only**.

## Features
- User login and registration pages  
- Job listings display  
- Job filtering by keyword and location  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## Usage
Open the HTML files (`login.html`, `signup.html`, `jobs.html`) directly in a web browser.

## Notes
This project is a front-end prototype and does not include backend authentication or database integration.
